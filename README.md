# 💰 Expense Tracker (Java Console App)

## 📌 Overview

A console-based Expense Tracker built using Core Java.
It allows users to record, categorize, and analyze daily expenses with persistent storage using a CSV file.

---

## 🚀 Features

* Add daily expenses
* View all expenses
* Filter by category
* Calculate total spending
* Persistent storage using CSV file
* Exception handling & input validation

---

## 🛠️ Tech Stack

* Core Java
* Java Collections (ArrayList)
* File I/O (BufferedReader, BufferedWriter)
* Exception Handling

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/expense-tracker.git
```

2. Navigate to project

```
cd expense-tracker
```

3. Compile

```
javac -d . src/com/expensetracker/**/*.java
```

4. Run

```
java com.expensetracker.main.ExpenseTrackerApp
```
